 xmlns="http://www.bml-initiative.org/bml/bml-1.0" 
        character="Alice" 
        id="bml1">
     id="speech1" start="6">This is a complete  id="syncstart1" /> 
    BML core speech description.

 
 
Example conversion to IndoGolog/Prolog
 
[bml, 
    [ xmlns, "http://www.bml-initiative.org/bml/bml-1.0], 
    [ character, "Alice" ], 
    [inner, 
        [locomotion,
              [id, "behavior1"],
              [target, "AUDIENCE"], 
              [manner, "WALK"]
        ]
    ]
]


A speak example



     Hello! This is a basic SSML BML test.
    
              
        Hello!  This is a basic SSML BML test.
      
    
  


[bml, 
    [ xmlns, "http://www.bml-initiative.org/bml/bml-1.0], 
    [ character, "Alice" ], 
    [inner, 
        [speech, 
              [id, "s1"],
              [inner, 
                  [text,
                      [inner, "Hello! This is a basic SSML " ]
                  [sync, 
                       [id, bml]
                  ],
                  [text, "BML test." ]
                  [description,
                      [priority, 2]
                      [type, "application/ssml+xml"]
                   
                  
              ], 
        ]
    ]
]

Going to be context sensitive and order matters.
